שם המטלה: נבחרת הכותבים

 

הדרישה:

 

עלייך לבנות רכיב מסוג "נבחרת הכותבים" אשר מכיל 5 תמונות של כתבים על מסך desktop ו 2.5 על מסך מובייל.

העיצוב של הרכיב יילקח מקובץ העיצוב המצורף כמו גם התמונות של הכותבים.


https://xd.adobe.com/view/3dd424c7-1fb4-4e3f-a195-4ccf3ace41ef-6887/


https://xd.adobe.com/view/1ef3dc2f-1d67-4e81-8353-984f92103645-4ba0/specs/

 

 

 

נדרש לכתוב שרת PHP ללא framework או שרת node.js אשר ממנו יימשכו הנתונים מה Client ע"י בקשת ajax בפורמט JSON.

ה JSON אשר יימשך מהשרת יכיל 3 שדות:

* ה URL של תמונת הכתב

* שם הכתב

* הטקסט שמתחת לתמונת הכתב

 

 

דגשים:

 

* לטובת פיתוח הקומפוננטה יתבצע שימוש בספריית React.js.

* ניתן להשתמש בכל ספריית JS ו\או CSS חיצונית.

* העיצוב צריך להיות pixel perfect

* יש להגיש את התוכנית בקובץ ZIP ולהוסיף קובץ README.md עם פרטים כיצד להפעיל את התוכנית (יש להגיש ללא node_modules)

*  יש לטפל בשגיאות ברמת הסביר (למשל אם ה client מקבל סטאטוס 404)

 

דגשי קוד:

 

* קוד נקי ללא ריווחים \ פסקאות מיותרות.

* אנגלית ללא שגיאות.

* שמות משתנים משמעותיים.

* להשתדל לא להשאיר שורות קוד ב quotes

* תיעוד סביר.

 

בונוס (רשות):

 

* לשלוף את המידע מטבלת MySQL בצד השרת./used google sheets instead

* להפוך את הרכיב לקרוסלה שמזיזה כל פעם תמונה אחת בלולאה אינסופית כאשר המשתמש לוחץ על החצים.
